## Getting Started
* npm
  ```sh
  npm install npm@latest -g
  ```
## Installation
1. Clone the repo
   ```sh
   git clone https://github.com/yang0709-dev/Habit-Tracker
   ```
2. cd to the "habit-tracker" folder
   ```sh
   cd habit-tracker/
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run the project
   ```sh
   npm run dev
   ```
